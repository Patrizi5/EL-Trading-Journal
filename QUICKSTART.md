# Eternum Trading Journal - Quick Start Guide

Get up and running with Eternum Trading Journal in 5 minutes!

## 1. Installation (Instant)

Simply open `index.html` in your web browser. That's it! No installation, no build process, no dependencies.

Supported browsers:

- Chrome/Chromium 90+
- Firefox 88+
- Safari 14+
- Edge 90+
- Mobile Safari (iOS 14+)
- Chrome Mobile (Android 90+)

## 2. Create Your Account

**Step 1: Click "Get Started Free"**

- Or click "Sign In" if you already have an account

**Step 2: Register or Login**

- Enter your first and last name
- Enter your email address
- Create a strong password (8+ characters recommended)
- Accept the terms of service

**Step 3: Explore Dashboard**

- You'll see sample trades for demonstration
- Overview of your performance metrics
- Recent trades and equity curve

## 3. Record Your First Trade

**Step 1: Go to Trading Journal**

- Click "Journal" in the navigation menu
- Or click "New Trade" button

**Step 2: Fill in Trade Details**

```
Market:              EUR/USD
Direction:           Long ‚Üë
Entry Price:         1.0850
Exit Price:          1.0880
Position Size:       1.0 lot
Stop Loss:           1.0800
Take Profit:         1.0920
Strategy:            Swing Trading
Timeframe:           1 hour
Pre-Trade Emotion:   Confident
Post-Trade Emotion:  Satisfied
Notes:               Strong support bounce, clean setup
```

**Step 3: Click "Save Trade"**

- Trade is automatically saved to your local storage
- All data stays on your computer - no cloud required

## FAQ

**Q: Is my data saved automatically?**
A: Yes! Data is saved to your browser's localStorage as you enter it.

**Q: Can I access my data from other devices?**
A: Not automatically. Export your data and re-import on other devices.

**Q: What happens if I clear my browser cache?**
A: Your data will be lost. Always keep backups!

**Q: Can I use this on mobile?**
A: Yes! Responsive design works on phones and tablets.

**Q: Is there a native app?**
A: Not yet. Mobile apps planned for version 1.1.

**Q: Can I trade while using the app?**
A: The app is for journaling only. Use your broker for actual trading.

**Q: Is there a cost?**
A: Eternum Trading Journal is completely free!

**Q: How often is it updated?**
A: Regular updates with new features and improvements.

## Troubleshooting

**Problem: Data not saving**

- Check if localStorage is enabled
- Ensure sufficient storage space
- Try refreshing the page
- Check browser console for errors

**Problem: Charts not loading**

- Check internet connection
- Verify TradingView configuration
- Try refreshing the page
- Use a different browser

**Problem: Slow performance**

- Clear browser cache
- Reduce number of trades in memory
- Disable background animations
- Try a different browser

**Problem: Lost data**

- Check if data is in localStorage
- Look for browser backups
- Restore from your backup file
- Contact support for assistance
